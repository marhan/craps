# A simple simulation of the dice game [Craps](https://en.wikipedia.org/wiki/Craps)

## Prerequisites

* JDK 8

## Test and run project 

### Gradle wrapper
  
    $ ./gradlew clean test run

### Local gradle installation
  
    $ gradle clean test run
